
## Autonomous Market Intelligence System

A production-ready multi-agent architecture built with CrewAI that autonomously generates 
executive-grade market research reports. Features intelligent LLM routing, sequential 
task orchestration, and cross-agent context sharing.

**Architecture Highlights:**
- 3 specialized agents with distinct roles and tool access
- 2-tier LLM strategy (8B for speed, 70B for quality) optimizing cost/latency
- Sequential pipeline with context injection between research, analysis, and writing phases
- Automated markdown report generation for C-level stakeholders
